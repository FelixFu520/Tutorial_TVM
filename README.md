```

3. 使用命令行界面编译和优化模型

5. 使用张量表达式操作算子
6. 使用模板和 AutoTVM 优化算子
7. 使用无模板的 AutoScheduler 优化算子
8. 交叉编译和远程过程调用（RPC）
9. 用 GPU 编译深度学习模型
```
